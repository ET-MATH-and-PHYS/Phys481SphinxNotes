IntroductionÂ¶

There are many interesting systems for which the variation of an important physical quantity \(u\) can be accurately modelled with a one dimensional wave equation

\[\frac{\partial^2 u}{\partial t^2} = c^2 \frac{\partial^2 u}{\partial x^2}, \quad x\in (0,L),\ t\in (0,T]\]

where \(c\) is a characteristic propagation speed.

This partial differential equation (PDE) is second order in time, so we require two sets of initial conditions in order to fully specify it. These are typically chosen to be some profile \(I(x)\) at \(t=0\)

\[u(x,0) = I(x), \quad x\in [0,L]\]

and the corresponding initial time derivative \(I'(x)\) also at \(t=0\).

\[\frac{\partial}{\partial t}u(x,0) = V(x), \quad x\in [0,L]\]

We often also need to consider spatial boundary conditions. These are usually located at the outer edges of some region of interest.

\[u(0,t) = 0, \quad t\in (0,T]\]
\[u(L,t) = 0, \quad t\in (0,T]\]

DiscretizationÂ¶

Numerical solutions to the wave equation can be obtained by replacing continuous derivatives with discrete differences. Using centered second differences

\[\frac{u_i^{n+1} - 2u_i^n + u^{n-1}_i}{\Delta t^2} = c^2\frac{u_{i+1}^{n} - 2u_i^n + u^{n}_{i-1}}{\Delta x^2}\]

provides an explicit expression for future values

\[u(x_i,t_{n+1}) = u^{n+1}_i\]

at any location in terms of current and previous values

\[u^{n+1}_i = -u^{n-1}_i + 2u^n_i + C^2 \left(u^{n}_{i+1}-2u^{n}_{i} + u^{n}_{i-1}\right)\]

Speed of informationÂ¶

It is important to note that this difference equation depends on a dimensionless parameter \(C\) which is a product of the intrinsic propagation speed \(c\) with the ratio of grid steps in time \(\Delta t\) and space \(\Delta x\).

\[C = c\frac{\Delta t}{\Delta x}\]

When \(C>1\) the largest effective numerical propagation speed is less than the physical speed, typically leading to inaccurate results. https://en.wikipedia.org/wiki/Courant%E2%80%93Friedrichs%E2%80%93Lewy_condition

Finally, when system is initially static \(I'=0\)

\[I' = \frac{u_i^{n+1} - u^{n-1}_i}{2 \Delta t} = 0\]
\[{u_i^{n+1} = u^{n-1}_i}\]

the first step doesnâ€™t require information about \(t<0\)

\[u_i^1 = u^0_i - \frac{1}{2} C^2\left(u^{0}_{i+1}-2u^{0}_{i} + u^{0}_{i-1}\right)\]

while it is only slightly harder to deal with more general non-zero initial first derivatives.

def wave_stepper(u, Courant=0.1, nsteps=99, save_stride=9, boundary_hack=False):
    """Explicit time steps of the dimensionless wave equation.
    """
    C2 = Courant**2
    u1 = 1*u.copy()  # current displacement
    u2 = 0*u.copy()  # future

    # spatial second derivative difference equation assuming
    # cyclic ("wrap-around") boundaries
    d2u = lambda u: np.roll(u,-1) - 2*u + np.roll(u,+1)

    u0 = u.copy()  #previous displacement if initial velocity is zero
    term2 = d2u(u1)
    u0 = u1 + 0.5*C2*term2

    result = []
    for n in range(nsteps):
        term2 = d2u(u1)
        u2 = 2*u1 - u0 + C2*term2

        if boundary_hack:
            u2[0], u2[-1] = 0, 0

        u0 = u1[:]  # previous = current
        u1 = u2[:]  # current = next

        # save results at every 9th time step for analysis and display
        if (n%save_stride)==0: result.append(u1.copy())

    result.append(u1)
    return result

def wave_solver(u, xstep, tstep, wave_speed=1.0, save_stride=1, boundary_hack=False):
    """ Explicit physical solution of the wave equation.
    """
    dt = tstep[1] - tstep[0]
    dx = xstep[1] - xstep[0]
    C = wave_speed * dt / dx
    if (C>0.95):
        print('Warning- large Courant number: '+str(C)[0:4])

    return wave_stepper(u, C, len(tstep)-1, save_stride=save_stride, boundary_hack=boundary_hack ), C

Energy conservationÂ¶

Solutions to the wave equation are conservative. The total energy at any given time will be the sum of two quadratic terms

\[E(t):= \frac{1}{2}\int_0^L \left[ u_t^2(x,t) + c^2 u_x^2(x,t) \right] dx\]

which correspond to the kinetic and potential energies respectively.

standing wavesÂ¶

Standing waves arise from an initial condition

\[u(x,0)= A \sin\left(\frac{\pi}{L}mx\right)\]

where \(m\) is an integer and A is a freely chosen amplitude. The corresponding exact solution as a function of \(x\) and \(t\)

\[{u_{\small\mbox{e}}}(x,t) = A\sin\left(\frac{\pi}{L}{m}x\right) \cos\left(\frac{\pi}{L}{m}{c}t\right){\thinspace .}\]

must satisfy the wave equation.

It is interesting to simulate a standing wave on [0,L] and determine the error in the simulation as a function of time and location. Any choices for \(A\) and \(m\) should work, but simple values (ie. \(A=1\) and \(m=1\)) might be best.
